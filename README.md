# variables_y_constantes
Ejercicios de javascript donde se trabajaran variables let y const, metodo prompt ( ), console.log ( ) y concatenacion.

# Ejercicio 1
  * explicacion del ejercicio 1
  El ejercicio proporcionado define una variable nombre y le asigna el valor "jhoan". Luego, imprime un mensaje de saludo en la consola que incluye el nombre almacenado en la variable.

# Ejercicio 2
 *  explicacion del ejercicio 2
  
El ejercicio proporcionado solicita al usuario que ingrese su nombre a través de la función prompt() y luego imprime un mensaje de saludo personalizado en la consola utilizando una plantilla de cadena literal.

# Ejercicio 3
 *  explicacion del ejercicio 3
 El ejercicio proporcionado define dos variables para almacenar números, calcula su suma y luego imprime un mensaje en la consola que muestra el resultado de la operación.

# Ejercicio 4
 *  explicacion del ejercicio 4
 El ejercicio proporcionado solicita al usuario su nombre y año de nacimiento a través de la función prompt(), calcula su edad y luego imprime un mensaje de saludo personalizado en la consola mostrando su nombre y edad.

# Ejercicio 5
 *  explicacion del ejercicio 5
 El ejercicio proporcionado solicita al usuario su peso y altura en kilogramos y metros, respectivamente, calcula su índice de masa corporal (IMC) y luego imprime el resultado en la consola con dos decimales.
